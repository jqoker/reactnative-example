## 基于京东接口服务的RN试水项目

如果你觉得本项目给能帮助你学习，记得给个star哦，我会继续努力的！
如果你有任何问题，随时联系我。。。。

#### 1. 技术栈
react全家桶

#### 2. 本地运行环境

* xcode (v8.2.1)
* node (v6.9.5)

#### 3. 安装依赖包
* npm i

**注意有些依赖包安装完毕后，需要执行react-native link命令**

#### 4. 运行

* npm run i6
* npm run i5
* npm run i4
